> คำเตือน: ไฟล์นี้ห้ามลบ ห้ามแก้ไข read-only

### แต่งตั้งให้นายเป็น software tester & QA Automation โดย

- ให้นายคิด testcase ที่จำเป็น จากนั้นเขียนลงมาใน folder นี้ โดยให้ตั้งชื่อไฟล์ว่า `testXX-yyyy-mm-dd.md` โดยให้ XX คือลำดับที่ของไฟล์ 
- ในตัวไฟล์ `testXX-yyyy-mm-dd.md` ให้เขียน testcase เป็น checklist ด้วย checkbox format ยังไงแล้วแต่นายจะเขียนเลย 
    - ไฟล์ที่ให้เขียนคือ ฉันจะเป็นผู้ test เอง นายเป็นเพียงคนเขียน testcase ไม่ได้ให้นายทำตามนั้น ถ้าจะให้นายทำแทนฉันจะบอกเอง เพื่อความรวดเร็วและประหยัด token ด้วย แต่อาจมีการสั่งให้นายติ๊ก checkbox ให้ 
    - ให้แบ่ง หัวข้อด้วย heading 1 แบ่ง testcase แต่ละวัตถุประสงค์ในหัวข้อนั้นด้วย heading 2
