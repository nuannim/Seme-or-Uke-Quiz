# การทดสอบการปรับปรุงคำถามข้อที่ 3 เรื่องเพื่อนซบไหล่ (Task Q3 Choice D Friendly Tweak Test)

การทดสอบนี้จัดทำขึ้นเพื่อยืนยันว่าคำถามข้อที่ 3 ได้ถูกปรับปรุงตัวเลือก D ให้เป็นมิตรภาพทั่วไปของเพื่อน ไม่มีความโรแมนติกหรือการสัมผัสทางกายที่ลึกซึ้งเกินควร และมีคะแนนสะสมตรงตามที่กำหนด

---

## 1. ตรวจสอบเนื้อหาและคะแนนของคำถามข้อที่ 3 ในข้อมูลแบบทดสอบ

- [ ] เปิดไฟล์ข้อมูลแบบทดสอบ `public/data/qna3.js` หรือตรวจสอบข้อมูลใน `app.js` (fallback)
- [ ] ตรวจสอบว่าคำถามข้อที่ 3 มีเนื้อหาดังนี้:
    - คำถาม: `"ถ้ามีเพื่อนเอาหัวมาซบไหล่ คุณจะ..."`
    - Choice A: `"นั่งเฉย ๆ ไม่มีไร"`, `seme: 0, uke: 3`
    - Choice B: `"เอียงหัวไปพิงกลับเบา ๆ"`, `seme: 1, uke: 2`
    - Choice C: `"แซวขำ ๆ ว่าคิดค่าเช่าไหล่นะ"`, `seme: 2, uke: 1`
    - Choice D: `"ถาม 'ว่าไงจ๊ะ มีอะไรรึ'"`, `seme: 3, uke: 0`
- [ ] ตรวจสอบว่า Choice D ไม่มีคำว่า "เนียนโอบไหล่กลับ" หรือ "ลูบหัวเขาเบา ๆ" อีกต่อไปเพื่อเน้นย้ำความเป็นเพื่อนตามความต้องการของผู้ใช้

---

## 2. ตรวจสอบการใช้งานจริงในหน้าเว็บแอปพลิเคชัน (Manual UI Walkthrough)

- [ ] เปิดเว็บแอปพลิเคชันผ่านเบราว์เซอร์
- [ ] ตอบคำถามไปจนกระทั่งพบข้อที่ 3: `"ถ้ามีเพื่อนเอาหัวมาซบไหล่ คุณจะ..."`
- [ ] ตรวจสอบการแสดงผลของตัวเลือก Choice D: `"ถาม 'ว่าไงจ๊ะ มีอะไรรึ'"`
- [ ] ตัวอักษรและวรรคตอนแสดงผลถูกต้อง ไม่มีตัวสะกดผิดหรือเครื่องหมายอัญประกาศคู่ซ้อนกันจนแสดงผลพัง
- [ ] ทดลองคลิกเลือก Choice D และทำแบบทดสอบให้จบ เพื่อดูว่าไม่มีข้อผิดพลาดทางเทคนิคเกิดขึ้น
