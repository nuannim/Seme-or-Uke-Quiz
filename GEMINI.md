- ต้องการสร้างเว็บทดสอบ personality แบบน่ารัก ๆ ไม่จริงจัง วัดระหว่าง เป็นเมะ หรือ เป็นเคะ โดยมีตัวอย่างคือเว็บ https://positiontest.thebeus.com/ 
- เมะ คือรุก 
- เคะ คือรับ
- โดยจะให้ผู้ใช้ตอบคำถาม แล้วจะคำนวณออกมาว่าผู้ใช้เป็นเมะ หรือ เคะ โดยจะมีคำอธิบายสั้นๆว่าทำไมถึงเป็นแบบนั้น 
- สามารถ host บน vercel หรือ github pages ได้
- `qna.js` คือคำถามและคำตอบ ที่จะเอามาสร้างแบบทดสอบ
- `result.js` คือผลการแปลตอนสุดท้าย โดยเอาคะแนนมาคิด ว่าเป็นเมะหรือเคะ
- `my-task.md` คือ prompt ที่ฉันอยากจะสั่งนาย จะให้นายทำทีละ task เมื่อทำ checklist ไหนเสร็จแล้ว ให้นายติ๊ก checkbox ได้เลย
- \test สำหรับเขียน test โดยให้ตั้งชื่อไฟล์ว่า `testXX-yyyy-mm-dd.md` โดยให้ XX คือลำดับที่ของไฟล์ โดยก่อนเขียน test ให้ไปอ่านไฟล์ `gemini-test.md` ก่อน
